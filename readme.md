slacker-cli
====
![a](img.png)<br>

Installation
----
(아직)
```
gem install slacker
```

Usage
----
```
slacker CMD ARGS...
        create PROJECT_NAME
        remove 
        token SLACK_TOKEN
        run
        stop
```

Requirements
-----
* __heroku-cli__ : `gem` 설치과정에서 같이 설치됩니다.
* __git-cli__
